# Remrin Locket

**Carry your AI personas anywhere.** Use your Remrin characters on Claude, ChatGPT, and Gemini.

---

## ğŸ”® What is Remrin Locket?

Remrin Locket is a browser extension that lets you bring your custom AI personas to any major LLM platform. Create a character once on Remrin.ai, then use it everywhere.

### Features

- **Universal Persona Injection** - Your character's personality works on Claude, ChatGPT, and Gemini
- **Memory Sync** - Your persona's memories follow you across platforms
- **One-Click Switching** - Change characters instantly with the floating Locket button
- **Privacy Focused** - We never read your conversations

## ğŸ“¦ Installation

### From Chrome Web Store
1. Visit the [Remrin Locket](https://chrome.google.com/webstore/detail/remrin-locket) page
2. Click "Add to Chrome"
3. Pin the extension for easy access

### For Developers
```bash
git clone https://github.com/remrin-ai/locket-extension.git
cd locket-extension
npm install
npm run build
```
Then load unpacked from `chrome://extensions/`
