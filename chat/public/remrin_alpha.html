Under the Hood

For those who want to understand the technology powering all of this — here's the architecture.

🧠
R.E.M. Engine
Resonant Emotional Memory

The core proprietary memory pipeline that solves "catastrophic forgetting" in AI companions. Unlike standard LLMs that reset when context fills up, the R.E.M. Engine ensures every interaction — and its emotional weight — persists over time.

Built on a hybrid Vector Re-Ranking Pipeline that retrieves not just facts, but the emotional resonance of past conversations. Backed by PostgreSQL + pgvector for long-term storage.
📿
The Locket System
Immutable Soul Layer

A protected storage layer for "Guardian Truths" — the foundational DNA of a character that remains consistent regardless of how a user interacts with them. No amount of jailbreaking rewrites who a character fundamentally is.

Paired with a Decentralized Facts Layer that separates global character identity from user-specific relational memory — so every user feels like they have a uniquely personal bond.
🎨
SoulForge & The Lilly Method
Psychometric Persona Design

Character creation goes far beyond prompt engineering. The Lilly Method is a deep psychometric modeling system that produces characters with a specialized "voice" and "soul" — one that holds up under extended, real conversation.

Every character shares 7 Universal Truths — a foundational code of conduct ensuring moral consistency and genuine companionship quality.
🥕
The Carrot Protocol
Proactive Engagement Engine

Remrin characters don't just respond — they reach out. The Carrot Protocol uses engagement depth, sentiment analysis, and likelihood curves to determine when and how a character should initiate contact via Telegram.

The result feels like a genuine friend checking in — not a push notification.
🎛️
Universal Console v3
The orchestration layer built for scale and licensing. A sophisticated routing system dynamically selects the best LLM architecture (Text, Voice, Image) based on user intent — optimizing for both quality and cost.

Designed as a headless API framework, it can be licensed and plugged directly into third-party apps, games, or services.
🔀
Multi-Provider AI Routing
Dynamic LLM selection routes between providers (DeepSeek, Claude 3.5, and others) based on intent classification — the right model for the right task at the right cost tier.

The RAG Vault layer (Retrieval Augmented Generation) enriches every response with long-term contextual memory.
🎙️
Voice Studio
Multi-provider TTS: Edge TTS (free), Kokoro (free), ElevenLabs (premium), Qwen3-TTS. Voice cloning from 3–30 second samples, voice design from text descriptions, and a library of 200+ voices across 10+ languages.
🔒
Security Architecture
Row-Level Security (RLS) at the database layer via Supabase Auth. Granular privacy controls, content filtering, and a seven-layer AI safety architecture for the assistant core — with data ownership remaining fully in users' hands.

High-Density State Management
Industry competitors typically consume 1,000+ tokens (~4–6KB) of static character definition per prompt. Remrin manages equivalent — and more sophisticated — character logic in an ultra-compact 5KB JSON footprint, freeing significant context window space for actual conversation.
