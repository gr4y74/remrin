## Unit Economics

### Average Revenue Per User (ARPU)
- **Free users:** $0.50-2/month (credits, gacha)
- **Paid users:** $15-40/month (subscription + add-ons)
- **Creators (Architect/Titan):** $50-200/month (sales + consumption)

### Customer Acquisition Cost (CAC)
- **Organic (current):** ~$0-5/user
- **Paid marketing:** $10-30/user (estimated)
- **Payback period:** 1-3 months for paid users

### Lifetime Value (LTV)
- **Casual user:** $50-100 (6-12 months)
- **Engaged user:** $200-500 (12-24 months)
- **Creator/whale:** $1,000-5,000+ (24+ months)

**LTV:CAC Ratio:** 5:1 to 20:1 (excellent unit economics)

## Risk Mitigation

### Technical Risks
- **API cost increases:** Diversify providers (DeepSeek, local models)
- **Downtime:** Multi-provider fallback system already implemented

### Market Risks
- **Competition:** First-mover advantage in creator marketplace
- **Regulation:** Age verification, content moderation already in place

### Financial Risks
- **Churn:** Engagement features (memory, locket, shared facts) increase retention
- **Payment processing:** Stripe integration with fraud protection
